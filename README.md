# Overview

This project was developed as part of the **Artificial Intelligence & Machine Learning Internship** at **Main Flow Services and Technologies Pvt. Ltd.**

The chatbot was built in two stages:

- **Task 1:** Rule-Based Chatbot using Pattern Matching and Regular Expressions
- **Task 2:** NLP-Enhanced Chatbot using Machine Learning for Intent Classification

The chatbot can understand user queries related to admissions, fees, courses, placements, hostels, greetings, farewells, and general college-related FAQs.

---

# Key Features

- 🤖 Rule-Based chatbot using keyword and regex matching
- 🧠 NLP-powered intent classification
- 📚 TF-IDF Vectorization
- 🎯 Cosine Similarity based intent detection
- 📊 Logistic Regression classifier
- 📝 Text preprocessing
  - Lowercasing
  - Tokenization
  - Stopword Removal
  - Lemmatization
- 💬 Interactive command-line chatbot
- 📂 Knowledge Base driven responses
- 📈 Confusion Matrix visualization
- 💾 Model saving using Pickle (.pkl)
- ⚡ Fast intent prediction
- 🛡️ Graceful handling of unknown user queries
